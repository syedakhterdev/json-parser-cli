# Json Parser CLI (Node.js)

### Installation

Install the dependencies.

```sh
$ npm install
$ npm link
$ json-parser-cli
Please enter selecto
```
### Input Types
For class:
```sh
StackView
```
For classNames:
```sh
.container
```
For identifier:
```sh
#videoMode
```
For Compound selectors && Chain selectors
```sh
StackView.accessoryView
Input#videoMode
```
To exit from program:
```sh
exit
```
OR
```sh
ctrl + c
```
